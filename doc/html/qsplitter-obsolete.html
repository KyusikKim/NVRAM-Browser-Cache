Obsolete Members for QSplitter

The following class members are obsolete. They are provided to keep old source code working. We strongly advise against using them in new code.

  • QSplitter class reference

Related Non-Members

QTextStream & operator<< ( QTextStream & ts, const QSplitter & splitter ) (obsolete)
QTextStream & operator>> ( QTextStream & ts, QSplitter & splitter ) (obsolete)

Related Non-Members

QTextStream & operator<< ( QTextStream & ts, const QSplitter & splitter )

Use ts << splitter.saveState() instead.

QTextStream & operator>> ( QTextStream & ts, QSplitter & splitter )

Use ts >> splitter.restoreState() instead.
