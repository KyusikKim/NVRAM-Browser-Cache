Toys: Clocks Example

Files:

  • declarative/toys/clocks/clocks.qml
  • declarative/toys/clocks/content/Clock.qml
  • declarative/toys/clocks/content/QuitButton.qml

This example displays a set of clocks with different times for different cities. Each clock is created by combining Image elements with Rotation transforms and SpringAnimation behaviors.
