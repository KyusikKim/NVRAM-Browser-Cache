makeqpf

makeqpf is a tool to generate pre-rendered fonts in QPF2 format for use on Embedded Linux.

Qt 4 can read files in QPF2 format in addition to QPF files generated by older versions of makeqpf from Qt 2 or 3.

See also Qt for Embedded Linux Fonts.
